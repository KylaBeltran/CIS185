Skincare Picks
CIS 185 – Midterm Project
Author: Kyla Andrea Beltran
Date: 11/14/2025

1. Project Description

Skincare Picks is a curated product showcase website featuring popular skincare items from Korea, Japan, and the USA.
The site allows users to browse products, filter by country, and filter by product type.
The goal is to provide a clean and aesthetic catalog for people discovering new skincare favorites.

2.Target Audience

This website is designed for:

Skincare beginners looking for simple product recommendations

Beauty enthusiasts interested in Korean, Japanese, and American skincare

Anyone trying to compare products by category or skin concern


3. Main Features

Product Grid Display — 27 curated skincare products with images and descriptions

Country Tabs — Filter products by Korea, USA, Japan, All

Product Type Filters — Toner, SPF, Serum, Moisturizer, Essence, Cleanser, Mask

Responsive Layout — Works on desktop, tablet, and mobile

Aesthetic UI — Pink theme with side borders

Contact Form — Users can submit questions or requests

About Page — Describes the purpose and vision of the site

Clean Navigation Bar — Links to Home, About, and Contact
